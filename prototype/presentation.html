Website Redesign Proposal

Church of Saint Stephen-in-the-Fields

Prepared by Colin · March 2026

01

Why we're doing this

The current website runs on Textpattern CMS and requires someone who knows HTML to make even simple updates—changing service times, posting announcements, or updating contact information.

The redesign has two goals: modernize the look and feel so the site is welcoming and easy to navigate (especially on phones), and move to a platform anyone can maintain without touching code.

05

What stays, what changes

The redesign keeps everything familiar. The structure and content are the same—the changes are visual and technical.

Staying the same

  • Same six navigation sections
  • Same welcome message and imagery
  • Same sidebar: services, donations, social links
  • Same programs highlighted on the homepage
  • Same contact information and structure
  • Affirming Congregation badge

What's new

  • Clean, modern typeface (Public Sans)
  • Warm golden colour palette
  • Church photo in the header
  • Proper mobile layout
  • Alert box for seasonal notices
  • New CMS—no HTML knowledge needed to edit
06

Navigation structure

The existing site's six main sections are preserved. The current dropdown sub-menus have been removed in favour of a flat structure—secondary pages are accessed from within each section. This is simpler to use and much easier to maintain.

  • Home
  • Contact
  • Worship
  • Outreach
  • Arts
  • History
